# FileAI 安装使用说明

## 简介

FileAI 是一款 AI 驱动的 macOS 文件整理工具，能自动扫描你的 Downloads、Desktop、Documents 文件夹，通过 AI 分析文件内容并智能归类整理。

---

## 安装步骤

### 第一步：安装应用

1. 双击 `FileAI_1.4.0_aarch64.dmg` 打开安装包
2. 将 **FileAI** 图标拖入 **Applications（应用程序）** 文件夹

### 第二步：解除 macOS 安全限制（必须执行）

由于应用未经 Apple 官方签名，macOS 会默认阻止运行。打开**终端**（在启动台搜索"终端"），粘贴以下命令并按回车：

```
xattr -cr /Applications/FileAI.app
```

### 第三步：配置 AI API Key

FileAI 需要一个 AI 大模型的 API Key 才能工作。推荐使用 **DeepSeek**（费用极低，约 ¥0.001/次）。

**获取 DeepSeek API Key：**
1. 访问 [https://platform.deepseek.com](https://platform.deepseek.com) 注册账号
2. 充值（最低 10 元即可使用很久）
3. 在「API Keys」页面创建一个 Key，复制保存

**填入 API Key：**
1. 打开 FileAI，点击右上角 **设置（齿轮图标）**
2. 在「AI 配置」中选择 **DeepSeek** 模型
3. 将复制的 API Key 粘贴进去，点击保存

---

## 使用方法

1. **扫描文件**：点击「开始扫描」，FileAI 会扫描 Downloads、Desktop、Documents 中的文件
2. **AI 分类**：扫描完成后点击「AI 整理」，等待 AI 分析并为每个文件建议目标文件夹
3. **确认执行**：检查分类结果，点击「确认执行」，文件将自动移动到 `~/Organized/` 目录下

---

## 常见问题

**Q：打开应用提示"文件已损坏"？**  
A：请确认已执行第二步的终端命令 `xattr -cr /Applications/FileAI.app`。

**Q：AI 分类失败，提示 API 错误？**  
A：请检查 API Key 是否填写正确，以及账户是否有余额。

**Q：整理后的文件在哪里？**  
A：所有文件默认整理到 `~/Organized/` 文件夹（即你的用户目录下的 Organized 文件夹）。

**Q：支持哪些 macOS 版本？**  
A：需要 macOS 10.15（Catalina）及以上版本，推荐 macOS 12+。
